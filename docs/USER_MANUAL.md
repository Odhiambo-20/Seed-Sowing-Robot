## 2. Safety Guidelines

### 2.1 Before Operation
- Inspect robot for damage
- Check battery charge level
- Verify seed and water levels
- Clear field of obstacles
- Establish GPS base station

### 2.2 During Operation
- Maintain safe distance from robot
- Monitor via mobile app
- Use emergency stop if needed
- Avoid direct contact with moving parts

### 2.3 After Operation
- Power down properly
- Clean seed dispenser
- Drain water system if needed
- Store in dry location
- Charge batteries

## 4. Operating the Robot

### 4.1 Pre-Operation Checklist
□ Battery charged (>80%)
□ Seeds loaded
□ Water tank filled
□ GPS signal strong (>8 satellites)
□ Mobile app connected
□ Field boundaries set
□ Emergency stop tested

### 4.2 Starting a Planting Session
1. Power on robot
2. Open mobile app
3. Select "New Session"
4. Define planting area
5. Set planting parameters
6. Review mission plan
7. Start operation

### 4.3 Monitoring Operation
- View real-time location on map
- Check battery status
- Monitor planting progress
- Review environmental data
- Respond to alerts

### 4.4 Pausing and Resuming
- Tap "Pause" in app
- Robot completes current row
- Stops safely
- Resume from last position

### 4.5 Emergency Stop
- Press hardware E-STOP button
- Or tap E-STOP in mobile app
- Robot halts immediately
- Investigate before resuming

## 6. Troubleshooting

### 6.1 Common Issues

#### Robot won't start
- Check battery charge
- Verify power connections
- Check emergency stop
- Restart robot

#### Poor GPS accuracy
- Check base station
- Verify RTK connection
- Clear sky visibility
- Wait for satellite lock

#### Seed dispenser malfunction
- Check seed level
- Clean dispenser
- Verify motor operation
- Check for blockages

#### Communication loss
- Check WiFi/Bluetooth
- Verify mobile app connection
- Check signal strength
- Restart communication

#### Obstacle detection false alarms
- Clean sensors
- Check sensor alignment
- Adjust sensitivity
- Verify mounting

### 6.2 Error Codes
- E001: Low battery
- E002: GPS signal lost
- E003: Obstacle detected
- E004: Seed dispenser jam
- E005: Water pump failure
- E006: Motor driver error
- E007: Sensor malfunction
- E008: Communication timeout
